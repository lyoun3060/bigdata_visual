 회귀분석은 종속 변수와 독립 변수 간의 관계를 분석하는 방법

 종속 변수는 예측하려는 변수 또는 관심 변수로, 다른 변수에 의해 영향을 받는 변수
 독립 변수는 종속 변수에 영향을 미치는 변수로, 종속 변수의 변화를 설명하기 위해 사용


OLS는 선형 회귀 분석의 일반적인 방법론 중 하나
	1. R-squared(결정계수) : 종속 변수의 변동성을 얼마나 설명하는지를 나타내는 지표
	R-squared 값은 모델의 설명력이 좋다는 것을 나타내며, 낮은 값은 모델이 종속 변수의 변동성을 잘 설명하지 못한다는 것을 의미
	
	2. F-statistic : F-통계량이 클수록 회귀 모델이 유의미한지를 나타냅니다
	 F-통계량이 243.3으로 나타났고, p-value가 0.00로 매우 작은 값이므로 회귀 모델 전체가 통계적으로 유의미
	
	3. coef와 P>|t| -> coef(계수), P>|t|(p-value)
	각 독립 변수의 계수(coef)는 해당 독립 변수의 종속 변수에 대한 영향력을 나타냄
	 p-value는 해당 독립 변수의 유의성을 검정하는 지표로, 작을수록 해당 독립 변수가 통계적으로 유의미한 영향을 미친다는 것을 의미
	
	
	4. 0.025와 0.975는 회귀분석 결과에서 계수(coef)의 신뢰구간(confidence interval)
	0.025랑, 0.975 95%의 신뢰구간에서의 하한값과 상한값을 나타내는것 
	따라서 해당 값 사이에 있을것이다라고 추측가능(95%의 신뢰구간에서)
	
	※ 잔차(residual) : 회귀 분석에서 종속 변수와 예측된 값 사이의 차이를 의미
	5. Omnibus : Omnibus 검정은 잔차들이 정규 분포(종모양 그래프)를 따르는지를 검정하는 통계량
	Omnibus 값이 클수록 잔차가 정규 분포를 따른다는 가정에 어긋날 가능성이 커짐
	
	
	6. Prob(Omnibus) : Prob(Omnibus)는 Omnibus 검정의 유의 확률(p-value)입니다.
	유의 확률이 작을수록 잔차가 정규 분포를 따르지 않는다는 것을 의미
	
	7. Skew : Skew는 잔차의 왜도(skewness)를 나타내는 지표입니다. 왜도는 분포의 비대칭 정도를 나타냄
	
	8. Kurtosis : Kurtosis는 잔차의 첨도(kurtosis)를 나타내는 지표입니다. 첨도는 분포의 뾰족한 정도를 나타냄
	첨도 값이 3에 가까울수록 분포가 정규 분포에 가깝다는 것을 의미
	여기서 첨도 값이 4.095로 나타났으므로, 잔차의 분포는 정규 분포보다 더 뾰족한 형태
